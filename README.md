# Financial Fraud Detection Dashboard

An AI-powered financial analytics dashboard that detects suspicious transactions using anomaly detection and visualizes transaction insights interactively.

## Features

- Upload transaction CSV datasets
- Fraud and anomaly detection
- Interactive transaction analytics
- Suspicious transaction monitoring
- Real-time dashboard
- Plotly-based visualizations

---

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Scikit-learn
- Isolation Forest

---

## How It Works

1. User uploads transaction dataset
2. System preprocesses numerical transaction features
3. Isolation Forest identifies anomalous transactions
4. Dashboard visualizes transaction distribution
5. Suspicious transactions are highlighted dynamically

---

## Features Implemented

- Fraud Detection Pipeline
- Anomaly Detection
- Interactive Analytics Dashboard
- Real-Time Visualization
- Suspicious Transaction Reporting
